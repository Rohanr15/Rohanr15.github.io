Tyre Modelling and Suspension Tuning

Developing a lateral brush tyre model and tuning suspension geometry to mitigate understeer in high lateral acceleration scenarios.

Objective

Part 1: Develop a tyre model to predict cornering force as a function of lateral slip using a brush model approach.
Part 2: Optimise suspension geometry to reduce understeer at 1g lateral acceleration by balancing camber angle and toe.

Part 1 – Tyre Modelling

  • Constructed a lateral brush model using an exponential decay friction approach.
  • Model parameters were tuned to match experimental cornering force curves.
  • Implemented in MATLAB for iterative simulation and validation.

The resulting model accurately captured tyre behaviour beyond the critical slip point and was shown to be adaptable to varying normal loads.

Part 2 – Suspension Geometry Optimisation

  • MATLAB used to automate optimisation of suspension pickup points using fminsearch.
  • Objective function minimised understeer while preserving acceptable camber/toe characteristics.
  • Geometry changes visualised using plots and simulation feedback loops.

The optimisation process improved cornering balance and reduced the need for manual suspension tuning.
